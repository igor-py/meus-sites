Proposta do Site

Aqui colocarei o site que estou desenvolvendo, será um site pessoal com Softwares que eu mesmo desenvolver.
Tudo isso utilizando conhecimentos de Front-end e Back-end que aprendi no processo.


Na parte do CSS da página procurei utilizar Sass e Compass. Além disso, utilizei FlexBox para posicionar os elementos da melhor forma possível.


O site será totalmente responsivo, ou seja, adaptado para mudar conforme acessado por um dispositivo com uma tela menor que a de um computador.


Comecei com a linguagem de programação Python, aonde desenvolvi uma calculadora e um jogo de advinho, utilizando tkinter. Além disso, no meu primeiro estágio fiz um software que lia um arquivo em PDF e retirava apenas as informações pertinentes, utilizando a biblioteca de Regex.


No segundo estágio, tive mais contato com banco de dados, mais especificamente PostgreSQL, aonde aprendi a filtrar dados nos bancos relacionais e montar gráficos e relatórios estatísticos.

Projetos

Calculadora


Projeto em que o professor pediu que fosse feita uma Calculadora utilizando Python 3 e a biblioteca tkinter, utilizada para criar interfaces visuais em Python. Utilizei o paradigma da programação orientada a objetos OOP, separando em duas classes: Uma para renderizar a Calculadora e outra que cuidava de todos os cálculos das expressões que o usuário colocar.


Dentre outras funcionalidades estão o suporte à input colocado diretamente pelo teclado ou utilizando o mouse e clicando nas teclas da calculadora, além de aceitar expressoẽs matemáticas. Caso queira olhar o código basta acessar o repositório no Github

Professor Online


Projeto em que desenvolvi com amigos da faculdade, tivemos que criar uma solução durante os tempos de quarentena na vertente de educação. Então resolvemos criar uma plataforma em que um aluno com dúvidas poderia se cadastrar em busca de professores daquele determinado assunto. Enquanto, um professor voluntário poderia se cadastrar e colocar qual disciplina gostaria de ensinar.


Usamos as seguintes ferramentas para a criação do site: React, Styled-components, NodeJS e CSS


Ao lado temos o vídeo da plataforma que desenvolvemos durante o Hack-A-thon. Caso queira olhar o código ou ler a documentação do projeto basta acessar o repositório no Github
