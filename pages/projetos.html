Calculadora

Projeto de Calculadora utilizando a linguagem Python, para a interface gráfica foi utilizada a biblioteca Tkinter. Nesse projeto procurei utilizar Classes para praticar o paradigma da programação orientada à objeto, além de separar uma classe para a lógica da calculadora e outra para renderizar.

Guess Game

Projeto de um jogo de advinho, no qual o jogador tem que acertar um número de 1 a 100. O jogador tem direito à 10 tentativas e a cada chute, é impresso na tela, uma dica se o jogador está perto ou longe do número correto.
